# Public API snapshot

`flow.api` is export data generated by Go's `apidiff` tool. CI compares it with
the current module and rejects incompatible changes.

After an intentional breaking change before v1, regenerate it from the module
root with the pinned tool version used by CI:

```sh
go install golang.org/x/exp/cmd/apidiff@v0.0.0-20260709172345-9ea1abe57597
apidiff -m -w api/flow.api github.com/Tangerg/flow
apidiff -m -incompatible api/flow.api github.com/Tangerg/flow
```

After v1, do not update the snapshot merely to make CI pass. Preserve the old
API or release a new major version.
